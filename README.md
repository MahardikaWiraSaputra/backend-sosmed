[Nest](https://github.com/nestjs/nest) framework TypeScript starter repository.

## Installation

```bashnpm install
npm install prisma --save-dev
npx prisma generate
npx prisma migrate dev --name init
npm install @prisma/client
```

## Running the app

$ npm run start:dev
